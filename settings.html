Adjust the user settings below to customize the look and functionality of this website. User settings will not persist if using a guest or incognito window.

For best results, use a modern browser like Google Chrome and setup a user profile (See Add a person or profile on the chrome support page.) It is possible to have your settings persist across devices if you are using the same profile and browser to view the page.


Play video on homepage - This setting controls the download and display of the video on https://www.kcc.edu. The video is "on" by default for non-mobile users. Switching to "off" will prevent downloading and playing plus improve homepage loading speeds.

Dark mode - This setting is "off" by default. Turn dark mode "on" to enable a dark color theme that may be easier on the eyes.
